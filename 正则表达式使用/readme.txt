                                                       正则表达式概述

1.在正则表达式中，如果给出具体字符，就可以精确匹配，用 \d 可以匹配一个数字，\w 可以匹配一个字母或者数字。
 
    00\d 匹配 007 ,但是无法匹配00A
   \d\d\d 匹配  010,250
   \w\w\d 可以匹配'py3'
   . 可以匹配任意字符 


\d{3}\s+\d{3,8}

\d{3} 表示匹配 3 个数字，例如'010'；

\s 可以匹配一个空格（也包括 Tab 等空白符），所以 \s+ 表示至少有一个空格，例如匹配' '，' ' 等；

\d{3,8} 表示 3-8 个数字，例如'1234567'。
